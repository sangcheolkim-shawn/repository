# CMS 고도화 프로젝트 (뉴드림스 CMS Admin)

대교 콘텐츠 관리 시스템(CMS) 차세대 전환 프로젝트의 **인터랙티브 프로토타입과 설계 산출물**을 공유합니다.

- **담당**: 김상철 (플랫폼기획팀)
- **현재 버전**: 프로토타입 V9.9+ / Release v1.5
- **협업 채널**: Jira(CMS 프로젝트) · Confluence(NDCM 스페이스)

---

## 핵심 산출물

| 파일 | 설명 |
|------|------|
| `cms_stg_prd_v9.9.html` | **최신 프로토타입** — 13개 활성 뷰(작업 보드·등록·검수·배포·커리큘럼·양식·시스템 관리 등). 브라우저로 바로 열람 |
| `cms_stg_prd_release_v1.5.html` | 개발팀 전달용 릴리즈 본 |
| `cms_login.html` | 로그인·온보딩 화면 |
| `CMS_Wireframe_Descriptions.html` | 화면설계서(어노테이션) |
| `CMS_Operation_Protocol.html` | 운영 프로토콜 |
| `CMS_VersionUp_Deploy_Flow.html` | 버전업·배포 흐름 |
| `CMS_UX_Audit_Report_v9.9.html` | UX 점검 리포트 |
| `CMS_OptionSet_Design_v2.html` | 옵션셋(선택형 필드) 설계 |
| `DESIGN.md`, `대교 CMS 디자인 시스템.md` | 디자인 시스템 정의 |
| `CLAUDE.md` | 프로젝트 컨텍스트(구조·데이터·컨벤션) |
| `tasks/` | 진행/대기 작업, 데모 시나리오, 워크플로우 등 |

## 프로토타입 열어보기

별도 설치 없이 `cms_stg_prd_v9.9.html` 파일을 브라우저로 열면 됩니다. 좌측 사이드바에서 13개 메뉴를 이동하며 화면 흐름을 확인할 수 있습니다.

## 주요 설계 포인트

- **콘텐츠 독립성 아키텍처** — 콘텐츠를 외부 시스템 과목코드 종속에서 분리, 다채널 재사용·배포
- **상태 기반 워크플로우** — 배정 → 등록 → 검수 → 배포
- **권한 그룹(RBAC)** · 커리큘럼 버전 관리 · 외부 수신(상품·배치) 관리
- **유형 코드 마스터 + 양식 드롭다운**(CMS-104)

> 상세 기획·요구사항·정책은 Confluence(NDCM) 문서를 정본으로 합니다. 본 저장소는 프로토타입·화면설계 산출물 공유용입니다.
